# Capitals Quiz 🌍

A little geography game: you're shown a country and four possible capitals — pick the right one.

- 10 questions per round, with a live score and progress bar
- Instant feedback (and the correct answer revealed when you miss)
- Final result screen with a friendly message and **Play again**
- Language switcher: **English / Русский** (remembers your choice, shared with the main portfolio)
- Matches the look of the portfolio: soft pink palette + twinkling sparkle background

**Live:** https://fedoseevaa.github.io/Capitals_quiz_game/
**Portfolio:** https://fedoseevaa.github.io

## Files
- `index.html` — game page
- `game.js` — country data (EN/RU), quiz logic, and UI strings
- `game.css` — styles
- `sparkles.js` — animated background

Built as a fun add-on to my portfolio.
